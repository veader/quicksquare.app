Privacy Policy

Simple Version

Quicksquare does not record, collect, or retain any of your information.

The only time any of your information leaves your device while using Quicksquare is when necessary for communications with the Foursquare API.

Technical Bits

In order to communicate with the Foursquare API, Quicksquare must retain an OAuth token. This token is securely stored within the Keychain on the device. Quicksquare never transmits this token to any other server than those used to operate the Foursquare API.

Location Services is required to use Quicksquare. The location information provided by Location Services is never transmited to any other server than those used to operate the Foursquare API. Location Services is only used when the app is running in the foreground.

Usual Legal Jargon

Personally-Identifiable Information

Quicksquare does not record or retain any Personally-Identifiable Information outside of your device. The only information stored on the device is the OAuth token mentioned above.
